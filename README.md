# Texas Hold'em Poker

A Texas Hold'em Poker game built with Flet.

## Run the app

### uv

Run as a desktop app:

```bash
uv run flet run
```

Run as a web app:

```bash
uv run flet run --web
```

## How to play

- Click **New Game** to start a round
- You play as "You" against 3 AI opponents (Alice, Bob, Carol)
- Use the action buttons at the bottom to Fold, Check/Call, or Raise
- The best 5-card hand wins the pot at showdown
